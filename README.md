Descripcíon: Segundo ejercicio realizado en el marco de la Tecnicatura en Análisi de Sistemas. Introduciendo el uso de variables en Python.
Objetvo: Comprender qué es una variable y cómo utilizarla para almacenar y mostrar información en pantalla.
Apredizajes: Una variable es un espacio en la memoria que permite guardar un dato para poder utilizarlo más adelante en el programa. Es posible combinar variables con la funcíon print() para mistrar mensajes personalizados.
