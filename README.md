Descripción: Segundo ejercicio realizado en el marco de la Tecnicatura en Análisis de Sistemas. Introduce el uso de variables en Python.
Objetivo: Comprender qué es una variable y cómo utilizarla para almacenar y mostrar información en pantalla.
Aprendizajes: Una variable es un espacio en memoria que permite guardar un dato para poder utilizarlo más adelante en el programa.
Es posible combinar variables con la función print() para mostrar mensajes personalizados.

nombre="Giovana"
print("Hola mundo soy ",nombre)
print("Un gusto saludarte ",nombre)
